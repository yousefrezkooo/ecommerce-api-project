# Project Planning

## Goal

This API is the backend for a small online store. It lets shoppers browse
products and build a cart, and lets an admin manage the product catalog.
It's built with Node.js, Express, and MongoDB (Mongoose).

## User Stories

- As a shopper, I can view products.
- As a shopper, I can view a single product's details.
- As a shopper, I can create a cart and add items to it.
- As a shopper, I can view my cart with full product details.
- As an admin, I can add or edit products.
- As an admin, I can delete a product that's no longer sold.
